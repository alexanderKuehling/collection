Grundlagen

Diese Website dient zur Veranschaulischung eines Ergebnisses, dass durch ein künstliches neuronales Netz erstellt wird. Außerdem basiert sie auf einem ehemaligen Privatprojekt, welches zur Darstellung meiner Coding-Kompetenzen umgestaltet wurde. Das beiliegende Python script war mein Einstieg in den Bereich Neuronale Netze und Maschine Learning. Es war nie als praktisch anwendbares Program gedacht, sondern das Thema Aktie wurde nur gewählt, da in diesem Bereich die Datenbeschaffung recht einfach ist. Des weiteren sind Teile des Codes auskommentiert oder aus anderen Programmen übernommen, weil es nur zur Übung diente
